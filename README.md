## usage

This is an extension script who manage the color Image of gameobject from a ColorLibrary and EnumLibrary

Use this extension if you have :
- EnumLibrary
- ColorLibrary

Our stored Color in our libraries are usable by script, this extension allow you to set the Color of an Image via Inspector

Get the github HTTPS project url and load it in Unity Package Manager windows

in Unity : Windows -> Package Manager -> + -> Add package from git url

### Installation

1. Get the github HTTPS project url and load it in Unity Package Manager windows
in Unity : Windows -> Package Manager -> + -> Add package from git url
2. Clone the repo
3. Add ColorData script in your gameobject
4. Add ColorExtension script in your gameobject
5. Drag your EnumLibrary
6. Drag your ColorLibrary
7. Set if the library is in int or enum
8. Choose the value.
9. Enjoy
